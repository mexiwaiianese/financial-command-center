# Financial Command Center PRD

## Purpose
A local-first household financial command center.

## Custom assumptions
- Household gross income: $190,000/year
- Effective tax rate: 0.2%
- Tithing target: 10% of gross income
- PNC transactions are mortgage payments
- Walmart is Groceries & Household
- Costco is Transportation / Fuel
- HELOC includes legacy consolidated debt plus a small ADU remodel draw
- ADU remodel budget: about $17,000
- ADU expected rent: about $1,550/month starting September 2026

## MVP scope
- Import CSVs
- Normalize transactions
- Categorize transactions
- Display dashboard
- Track ADU metrics

## Future scope
- SQLite database
- User-editable rules
- Budget vs actual
- Net worth
- Debt snowball/avalanche
- Monthly executive review
